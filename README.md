# iWeather

#### iWeather is an app that displays live weather data from openweathermap.org. It displays the current day weather info and also that of the next five days.

## Language 
- Swift 5

## Target Device
- iPhone
- iPad

## Tools 
- Xcode 9.3 compatible

## Deployment info 
- iOS 14.5
